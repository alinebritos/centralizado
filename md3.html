Bem-vindo ao MD3

Esta interface usa tokens semânticos (Primary, Container, OnSurface) para criar consistência.

Card Elevado

Este card usa uma sombra suave para se destacar do fundo. Ideal para conteúdo arrastável ou clicável.

Card Outlined

Este card tem uma borda e nenhum preenchimento de sombra. Útil para listas ou grupos de informações.

Primary Container: O MD3 usa "Containers" para áreas de destaque menor que o botão principal, mas maior que o fundo.
